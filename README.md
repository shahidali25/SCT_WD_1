# Responsive Landing Page

This project is a responsive landing page with an interactive navigation menu that changes color when scrolling or hovering.

## Features
- Fixed navigation bar
- Scroll color change effect
- Hover animation on menu items
- Fully responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
